Prework Study Guide

HTML

  • HTML provides a structure for a webpage.
  • Think of HTML as the frame of a house.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • CSS provides style for a webpage.
  • Think of HTML as the siding and wallpaper of a house.
  • CSS defines the colors, aesthetics, visual layout, and organization.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • pwd: print working directory
  • ls: list directory contents
  • cd: change directory
  • git status: checks what branch we are currently on
  • git checkout -b [branch name]: creates a new branch and switches to it
  • git pull origin [branch name]: sync local workspace with git repo
  • git push origin [branch name]: push commits to repo
  • git add -A: add all changes to the staging area
  • git commit -m "[comment]": create a commit and add a comment in quotes
  • touch [filename.extension]: create file in working directory
